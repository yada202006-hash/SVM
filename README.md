# Support Vector Machine (SVM)
# 📘 Overview
This project demonstrates the implementation of a Support Vector Machine (SVM) for classification tasks. 
SVMs are powerful supervised learning models that find the optimal hyperplane separating data points of different classes with maximum margin.

1. Implementation using scikit‑learn

2. Supports linear, polynomial, and RBF kernels
